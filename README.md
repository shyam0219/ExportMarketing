### Updating Contact Information
Edit the contact details in `index.html`:
- Email addresses (search for "info@ecoexportindia.com")
- Phone numbers (search for "+91 98765 43210")
- Business hours in the contact section

### Changing Colors
Update color variables in `css/style.css` (line 11-25):
```css
:root {
    --primary-green: #2d5f3f;
    --secondary-green: #4a8c61;
    /* ... other color variables */
}
```

### Adding New Products
Add new product cards in the `products-grid` section of `index.html`:
```html
<div class="product-card">
    <div class="product-image">
        <img src="YOUR_IMAGE_URL" alt="Product name">
    </div>
    <div class="product-content">
        <h3>Product Name</h3>
        <p>Description...</p>
        <ul class="product-features">
            <li><i class="fas fa-check"></i> Feature 1</li>
        </ul>
    </div>
</div>
```

### Modifying Form Fields
Edit form fields in the contact section of `index.html` and update validation in `js/main.js` accordingly.
